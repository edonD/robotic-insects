# Module 07: Swarm Coordination — Requirements

## Questions to Answer

1. **Communication**: How do robots coordinate (acoustic, RF, optical)?
2. **Scalability**: How many robots can coordinate together?
3. **Emergent Behavior**: Collision avoidance, formation control, task allocation
4. **Reliability**: What happens if communication drops?

## Success Criteria

- Collision avoidance distance: > 10 cm
- Swarm size: > 5 robots
- Communication range: 1–2 meters
- Latency: < 50 ms
