# Software Requirements & Installation

This project requires four main software packages. You can install most of them for free.

---

## Installation Order

1. **01_python.md** — Python packages (required)
2. **02_ansys_cfd.md** — CFD solver for aerodynamics (required for Wave 2)
3. **03_fem_structures.md** — FEM solver for structural analysis (required for Wave 2)
4. **04_gdstk_layout.md** — Layout generation for MEMS masks (required for Phase 2)

Follow each file in order and run the verification commands before proceeding.

---

## Quick Reference

| Software | Purpose | Cost | Difficulty |
|---|---|---|---|
| Python 3.9+ | Core simulations | Free | Easy |
| NumPy, SciPy, Matplotlib | Scientific computing | Free | Easy |
| OpenFOAM or Fluent | Wing aerodynamics CFD | Free (OpenFOAM) or $ (Fluent) | Medium |
| Elmer FEM | Structural & thermal | Free | Medium |
| FreeCAD | 3D geometry | Free | Medium |
| gdstk | MEMS mask generation | Free | Easy |

---

## System Requirements

- **OS**: Windows 10/11, macOS 11+, or Linux (Ubuntu 20.04+)
- **RAM**: 8 GB minimum, 16 GB recommended
- **Disk**: 20 GB free space (FEM + CFD require ~10 GB)
- **Time to install**: 2–4 hours total
