# Module 08: Fabrication Plan — Requirements

## Questions to Answer

1. **MEMS Process**: Which foundry? (SiNE, OA Xiamen, others)
2. **Materials**: Silicon, polymer, composite?
3. **Assembly Steps**: How to assemble wings, actuators, electronics
4. **Yield & Cost**: Estimate unit cost and production yield

## Success Criteria

- Process maturity: TRL 6+ (prototype demonstration)
- Expected yield: > 50%
- Unit cost < $1000 (research phase)
- Production time: < 8 weeks per batch
